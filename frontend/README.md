# Frontend Dashboard

React-based dashboard for managing chaos experiments.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Dashboard will be available at http://localhost:3000

## Features

- Create chaos experiments
- View experiment history
- Real-time experiment monitoring (TODO)
- Resilience leaderboard (TODO)
